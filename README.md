<h2>📂 Proyectos Destacados</h2>

<h3>🛴 Urban.Scooter – Plataforma web y API para alquilar scooters</h3>

<p>
Validé la calidad de una aplicación web y móvil para el alquiler de scooters eléctricos. Analicé los requisitos funcionales, diseñé pruebas aplicando técnicas como <b>partición de equivalencias</b> y <b>valores límite</b>, y validé la integración de datos con <b>Postman</b> y <b>DevTools</b>, asegurando precisión en los flujos backend y front-end.
</p>

<p>
Este trabajo me permitió optimizar pruebas funcionales y de UI, documentar hallazgos en <b>JIRA</b>.
</p>

<h4>🌟 Resultados destacados:</h4>
<ul>
  <li>🧠 Analicé y diseñé un <b>diagrama de flujo</b> para validar entradas en formularios, optimizando la detección de errores.</li>
  <li>🧪 Diseñé y ejecuté <b>45 casos de prueba</b> funcionales y no funcionales para entornos <b>Web, Mobile y API</b>.</li>
  <li>⚙️ Validé <b>4 endpoints</b> (GET, POST, DELETE) utilizando <b>Postman</b>, asegurando la correcta comunicación con la API.</li>
  <li>🐛 Reporté y documenté <b>34 incidencias</b> en <b>JIRA</b>, priorizadas según impacto y severidad.</li>
  <li>💾 Verifiqué la integridad de datos mediante consultas <b>SQL</b> y reporté inconsistencias críticas en la <b>base de datos</b>.</li>
</ul>
